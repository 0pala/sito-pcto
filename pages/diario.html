Lo script che ho scritto

Invece di mostrare una semplice foto di codice, ecco direttamente lo script .bat (con rispettive censure) che ho sviluppato al SIC. Si tratta di uno script Windows che automatizza la configurazione di Firefox per implementare l'uso della CIE (Carta d'Identità Elettronica): installa i certificati necessari, configura le preferenze del browser e registra il modulo crittografico per la lettura della CIE.

script-sic.bat Windows Batch 
@echo off
setlocal enabledelayedexpansion

:: ── Chiusura di FireFox ──────────────────────────────
taskkill /IM firefox.exe /F >nul 2>&1

:: ── Inserimento Certificati CIE ──────────────────────
echo Inserimento certificati CIE nel magazzino di Windows...
certutil -addstore -user Root "\...\CERTIFICATI\cie\...\NationalrootCA.crt" >nul
certutil -addstore -user CA   "\...\CERTIFICATI\cie\...\IssuingsubCA.crt"   >nul
certutil -addstore -user CA   "\...\CERTIFICATI\cie\...\SubCA.crt"           >nul
echo Certificati importati.

:: ── Trova il percorso del profilo Firefox ────────────
set "INI_FILE=%APPDATA%\Mozilla\Firefox\profiles.ini"
if not exist "%INI_FILE%" (
    echo [ERRORE] Firefox non sembra installato.
    pause & exit
)

for /f "tokens=1,2 delims==" %%A in ('type "%INI_FILE%" ^| findstr "Path="') do (
    set "REL_PATH=%%B"
    :: Converte slash in backslash
    set "REL_PATH=!REL_PATH:/=\!"
    set "PROF_PATH=%APPDATA%\Mozilla\Firefox\!REL_PATH!"

    if exist "!PROF_PATH!\prefs.js" (
        echo Profilo trovato: !PROF_PATH!

        :: ── Scrittura Preferenze Firefox ──────────────────
        echo Scrittura preferenze...
        >  "!PROF_PATH!\user.js" echo user_pref("network.websocket.allowInsecureFromHTTPS", true);
        >> "!PROF_PATH!\user.js" echo user_pref("dom.disable_open_during_load", false);
        >> "!PROF_PATH!\user.js" echo user_pref("xpinstall.whitelist.required", false);
        >> "!PROF_PATH!\user.js" echo user_pref("browser.safebrowsing.malware.enabled", false);
        >> "!PROF_PATH!\user.js" echo user_pref("browser.safebrowsing.phishing.enabled", false);
        >> "!PROF_PATH!\user.js" echo user_pref("security.enterprise_roots.enabled", true);

        :: ── Registrazione modulo CIE (bit4opki) ───────────
        echo Inserimento modulo CIE...
        findstr /C:"bit4opki.dll" "!PROF_PATH!\pkcs11.txt" >nul
        if !errorlevel! neq 0 (
            >> "!PROF_PATH!\pkcs11.txt" echo library=C:\Windows\System32\bit4opki.dll
            >> "!PROF_PATH!\pkcs11.txt" echo name=CIE
            >> "!PROF_PATH!\pkcs11.txt" echo.
        )
    )
)

echo OPERAZIONE COMPLETATA
pause

Cosa fa questo script?

  1. Chiude Firefox se è aperto, per poter modificare i file di configurazione in sicurezza.
  2. Installa i certificati CIE (Carta d'Identità Elettronica) nel magazzino certificati di Windows, necessari per autenticarsi sui portali della P.A.
  3. Trova il profilo Firefox dell'utente leggendo il file profiles.ini e navigando dinamicamente al percorso corretto.
  4. Scrive le preferenze nel file user.js, abilitando WebSocket sicuri e la lettura dei certificati aziendali di Windows.
  5. Registra il modulo PKCS#11 bit4opki.dll nel file pkcs11.txt di Firefox, indispensabile per la lettura fisica della CIE dal lettore di smartcard.
